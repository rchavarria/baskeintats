# Baskeintats - Claude Code Instructions

Basketball Statistics SPA for tracking youth basketball games.

## Available Skills

Use these skills by referencing them with `@` or asking to "use skill X":

- **@.github/skills/html-to-game.md** - Convert HTML from FBM to Game objects

## Quick Commands

- `npm run build` - Build for production
- `npm run dev` - Start dev server

## Key Paths

- `src/data/games/` - Game data files (TypeScript)
- `src/data/teams.ts` - Team definitions
- `src/data/venues.ts` - Venue definitions
- `src/model/` - Zod schemas for validation
